# Overview

This is a simple game that I originally thought of in my first years of high school. It involves simple avoiding lines that fall from the sky.
The starting point of the lines and the speed progressively go up and get faster respectively. It a simple game that shows the ease of use of
ggez, the main libray used.

[Software Demo Video](https://youtu.be/RQ6YIUWqQXI)

# Development Environment

I used the Visual Studio IDE for the entire creation of the game.

Rust is a C/C++ like language, but with more memory managmaent and a focus on being easy to use. Ggez was the main library used in creating the
software. It includes many helpful tools to create 2d (and 3d) game components.
